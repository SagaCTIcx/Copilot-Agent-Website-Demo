# OxaR CX – Simple AI Hackathon Demo Page

A simple static HTML mock company website prepared for AI hackathon demos and GitHub Pages hosting.

The page is designed for:
- a clean public-facing company mockup
- easy embedding of a Microsoft Copilot Studio web agent
- fast testing without frameworks or build tools

## Project purpose

This repository contains a lightweight demo website for `OxaR CX`, a fictional company focused on AI-powered customer experience.

The main goal is to provide a simple public website where a Copilot Studio agent can be embedded and tested.

## Files

- `index.html` – the main website file
- `README.md` – this file

## Design

The page uses:
- a simple enterprise-style layout
- red, white, and gold color palette
- one clearly defined AI advisor embed area
